> 🚀 高效、智能的 RSS 监控与 Telegram 推送工具

## ✨ 主要特性

- 🔍 **智能筛选** - 支持按类别和关键词精确筛选
- 📱 **多群推送** - 同时推送到多个 Telegram 群组/频道
- 🚀 **Docker 部署** - 一键部署，支持多平台架构
- ⚡ **实时监控** - 可配置检查间隔，实时获取最新内容
- 🎯 **智能去重** - 避免重复推送，节省资源
- 🗂️ **自动清理** - 定时清理历史记录，防止存储膨胀
