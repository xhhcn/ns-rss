# NodeSeek RSS 监控工具

> 🚀 强大的 RSS 监控解决方案，支持多种筛选条件和 Telegram 推送

- 📡 **实时监控** - 自动获取 NodeSeek RSS 最新内容
- 🔍 **智能筛选** - 支持类别和关键词过滤
- 📱 **多平台推送** - 支持多个 Telegram 聊天群组
- 🐳 **Docker 部署** - 一键部署，开箱即用
- ⚙️ **灵活配置** - 丰富的环境变量配置选项
